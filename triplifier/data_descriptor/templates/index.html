Welcome to Flyover

Your Data F.A.I.R.-ification Tool - Bridging Data Incompatibilities

Transform your data into semantically interoperable Resource Description Framework (RDF) format through our guided three-step process

1. Ingest

Upload and process your datasets to prepare them for semantic annotation and interoperability.

  • CSV and PostgreSQL support
  • Data grooming
  • Automatic RDF conversion
  • Link multiple graphs

2. Describe

Provide metadata and descriptions for your data variables to enhance semantic meaning.

  • Variable categorisation
  • Unit specifications
  • Local definitions
  • Data type mapping

3. Annotate

Apply semantic annotations using standardised ontologies to make your data more F.A.I.R.

  • Ontology-based annotations
  • Value mapping to terminologies
  • Semantic interoperability
  • Increase data F.A.I.R.-ness
